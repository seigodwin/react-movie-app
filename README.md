# React Movie App

A modern movie browsing application built with **React** and **TypeScript**. Browse trending movies, manage your favorite movies, and enjoy a clean, responsive user interface powered by React Context and local storage.

## 🚀 Live Demo

**https://react-movie-app-rouge-seven.vercel.app/**

## Features

* 🎬 Browse a collection of movies
* 🔍 Search for movies by title
* ❤️ Add and remove movies from your favorites
* 💾 Favorites are persisted using Local Storage
* ⚛️ State management with React Context API
* 🖼️ Display movie posters, titles, ratings, and release dates
* 📱 Responsive and modern UI
* 🛠️ Built with React, TypeScript, and Vite

## Technologies Used

* React
* TypeScript
* Vite
* React Context API
* CSS

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/seigodwin/react-movie-app.git
```

2. Navigate to the project directory:

```bash
cd react-movie-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the local development URL displayed in your terminal.

## Build for Production

```bash
npm run build
```
