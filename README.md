# React Movie App

A modern movie browsing application built with **React** and **TypeScript**. This project showcases React fundamentals such as reusable components, props, state management, and TypeScript integration while providing a clean and responsive user interface for displaying movie information.

## Features

* 🎬 Browse a collection of movies
* 🖼️ Display movie posters, titles, and release dates
* ❤️ Favorite button for movies
* ⚛️ Built with React and TypeScript
* 📱 Responsive and clean UI

## Technologies Used

* React
* TypeScript
* Vite
* CSS

## Getting Started

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local development URL displayed in your terminal.
